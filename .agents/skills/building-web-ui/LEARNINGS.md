- [2026-06-25][source:inference][task:run-log-realtime-stream][probation] The single `/events` SSE stream now carries TWO frame shapes distinguished by `frame.type`: `run` (throttled state snapshot, coalesced to ≤1/120 ms, last frame replayed on connect, per-agent logs capped at 200) and `agent-stream` (`{agentId,channel,text}`, raw output firehose — NOT throttled, NOT replayed, one frame per coalesced line, mirrored to `console.log`). - [2026-06-25][source:user][task:openrouter-all-models-freetext][probation] FREE-TEXT model entry in the vanilla combobox (`client/app.js`): the user wanted to "type the model name" for any id, listed or not. Pattern — `customCandidate()` returns `{id:q,label:q,custom:true}` when the trimmed query is non-empty AND not an EXACT catalog id; `comboMatches()` appends it after the substring matches. Three follow-on edits make a custom id STICK: `syncModelInput` shows `S.modelId` verbatim when `modelById` misses; `refreshModelsAndKeys` preserves a non-empty `S.modelId` across key/provider refresh (only seeds a default when empty) instead of resetting to `models[0]`; and the provider-switch handler sets `S.modelId=''` so a typed id never leaks across providers. Capability badges replace hiding: `reasoning` (thinking) + a soft `no tools` warning (`combo__badge--warn`) when `m.tools===false`. `node --check` + a tiny Node mirror of `customCandidate`/`comboMatches` is enough to verify the logic with no browser.

- [2026-06-26][source:inference][task:simulation-mode][probation] `WebRunManager` drives a run through a tiny structural seam — `subscribe(state)` + `subscribeAgentOutput(chunk)` + `start():Promise<{runId,manifest:{errorReason?}}>` — NOT the concrete `Orchestrator`. - [2026-06-26][source:inference][task:simulation-mode][probation] Serving the SPA at a SECOND path (`/simulation`) needs an EXPLICIT `serveStatic(res, root, 'index.html')` route placed BEFORE the catch-all `GET !startsWith('/api/')` static handler — otherwise the catch-all does `serveStatic(root, 'simulation')` and 404s on the bare path. - [2026-06-26][source:inference][task:simulation-mode][probation] Make a state-machine engine testable WITHOUT timers/wall-clock: split one logical step into a public `advance()` and have `start()` merely `setInterval(()=>advance(), tickMs)`. Tests drive `advance()` in a loop to completion (cap iterations so a regression can't hang) and assert invariants every tick (e.g. `activeAgentCount <= concurrency`).
