- [2026-06-26][source:inference][task:changelog-complete-to-v3.0.0][probation] Auditing CHANGELOG COMPLETENESS to vX: a commit's version by git ANCESTRY can disagree with where the author DOCUMENTED it — and the author is often right, so DON'T blindly re-version present content.
