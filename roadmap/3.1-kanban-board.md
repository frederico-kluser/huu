# 3.1 Kanban Board

> Phase 3 — TUI (Act 2 → Act 3: Resolution)

The main view. A 5-column Kanban board showing every task's status at a glance with keyboard navigation.

## Checklist

- [ ] Ink app shell with tab navigation (K/L/M/C/B)
- [ ] 5-column Kanban: Backlog, Running, Review, Done, Failed
- [ ] Card component: task ID, name, agent icon, model, elapsed time, cost
- [ ] Header: current act, current beat, total cost
- [ ] Keyboard navigation (arrow keys to select cards)
- [ ] Auto-refresh from SQLite polling

## Dependencies

- 0.2 SQLite Schema (read beat_state, messages for task status)
- 2.2 Orchestrator Loop (provides data to display)

## Outputs

- `src/tui/App.tsx` — main Ink application shell
- `src/tui/components/KanbanBoard.tsx` — 5-column board
- `src/tui/components/Card.tsx` — individual task card
- `src/tui/components/Header.tsx` — beat sheet progress + cost
