# 2.4 Merge Workflow (Tier 3-4)

> Phase 2 — Orchestration (Act 2: Confrontation)

The advanced merge tiers. Handles conflicts that can't be auto-resolved: heuristic resolution and AI-powered conflict solving.

## Checklist

- [ ] Tier 3: `ours`/`theirs` heuristic (last-touch-wins + file ownership tracking)
- [ ] Tier 4: AI Resolver (send conflict to Claude with per-file conflict history)
- [ ] Conflict history tracking in SQLite (which files conflict frequently)
- [ ] Human escalation path (pause queue, notify TUI)

## Dependencies

- 1.3 Merge Workflow Tier 1-2 (base merge infrastructure)
- 0.2 SQLite Schema (conflict history storage)

## Outputs

- Extended `MergeManager` with Tier 3-4 resolution
- Conflict history tracking and per-file resolution patterns
- Human escalation mechanism (SQLite message + TUI notification)

## Technical Notes

- Tier 3 heuristic: agent who last touched the file gets priority
- Tier 4: Claude receives the conflict diff + history of previous resolutions for that file
- All Tier 4 resolutions are flagged for human review in the TUI
