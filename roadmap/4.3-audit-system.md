# 4.3 Audit System

> Phase 4 — Intelligence (Act 3: Resolution)

Trust but Verify. Complete logging of every action, post-session analysis, and cost reporting.

## Checklist

- [ ] Complete tool call logging (timestamp, agent, tool, params, result)
- [ ] Post-session audit report generation
- [ ] Suspicious action flagging (unusual patterns, unexpected file access)
- [ ] Cost reporting per session, per feature, per agent

## Dependencies

- 0.2 SQLite Schema (audit_log table)
- 1.1 Agent Runtime (tool call interception)

## Outputs

- `src/audit/logger.ts` — tool call logging middleware
- `src/audit/reporter.ts` — post-session audit report generation
- `src/audit/cost.ts` — cost tracking and reporting by agent/model/phase

## Technical Notes

- Every tool call logged: timestamp, agent_id, tool_name, params (sanitized), result_summary
- Post-session report flags: unusual file access patterns, high-cost operations, failed actions
- Cost tracking uses model pricing: Opus $15/MTok, Sonnet $3/MTok, Haiku $0.80/MTok
- Reports viewable in TUI Cost tab and exportable as markdown
