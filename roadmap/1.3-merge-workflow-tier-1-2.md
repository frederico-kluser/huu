# 1.3 Merge Workflow (Tier 1-2)

> Phase 1 — Single Agent Loop (Act 1 → Act 2)

The first two tiers of the 4-tier merge pipeline. Handles the easy cases: fast-forward and recursive auto-merge.

## Checklist

- [ ] FIFO merge queue (SQLite-backed)
- [ ] Tier 1: fast-forward detection and execution
- [ ] Tier 2: recursive merge with automatic conflict detection
- [ ] Pre-merge detection via `git merge-tree --write-tree`
- [ ] Merge result logging to SQLite
- [ ] Tests for clean merges and conflict detection

## Dependencies

- 0.2 SQLite Schema (merge queue, merge results)
- 0.3 WorktreeManager (worktree operations)

## Outputs

- `src/git/MergeManager.ts` — FIFO queue + Tier 1-2 resolution
- Test suite with real git repos testing clean merges and conflict detection

## Technical Notes

- `git merge-tree --write-tree` (Git 2.38+) detects conflicts without touching the working tree
- Merge on main must be serialized (mutex/semaphore on shared refs)
