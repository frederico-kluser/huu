# 1.1 Agent Runtime

> Phase 1 — Single Agent Loop (Act 1 → Act 2)

The execution engine for individual agents. Handles the full lifecycle: spawn worktree, inject context, run Claude API, collect results, cleanup.

## Checklist

- [ ] Agent definition format (TypeScript interface matching YAML frontmatter)
- [ ] Agent spawner: creates worktree, injects context, runs Claude API call
- [ ] Context preparation: read from scratchpad, build focused prompt
- [ ] Tool execution: map agent tools to actual implementations
- [ ] Result collection: capture output, write to SQLite messages
- [ ] Cleanup: remove worktree on completion or abort
- [ ] Abort support via AbortController

## Dependencies

- 0.2 SQLite Schema (messages table for result collection)
- 0.3 WorktreeManager (worktree lifecycle)

## Outputs

- `src/agents/runtime.ts` — agent spawner and lifecycle manager
- `src/agents/types.ts` — agent definition interfaces
- `src/agents/tools.ts` — tool implementations mapped to agent allowlists
