# 1.4 Basic CLI

> Phase 1 — Single Agent Loop (Act 1 → Act 2)

Minimal command-line interface before the TUI exists. Just enough to run and monitor a single agent.

## Checklist

- [ ] `huu run "task description"` — single agent execution
- [ ] `huu status` — show current state
- [ ] Structured console output (not TUI yet, just formatted logs)

## Dependencies

- 1.1 Agent Runtime (to execute agents)
- 1.2 Builder Agent (first agent to run)
- 1.3 Merge Workflow (to merge results)

## Outputs

- `src/cli.ts` — entry point with command parsing
- `bin/huu` — executable script
- Formatted console output with colors and status icons
